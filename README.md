# 任务确认板系统

一个支持实时同步的任务确认板，专为4人团队设计。

## 功能特点

- ✅ 110个任务格子（编号213-322）
- ✅ 四个用户各有不同颜色标识
- ✅ 所有人都可以点击任何未完成的格子
- ✅ 别人完成的格子不能被其他人点击
- ✅ 实时同步所有客户端状态
- ✅ 支持取消已完成的任务（仅限自己完成的）
- ✅ 实时进度统计
- ✅ 数据持久化保存

## 用户分配

| 用户 | 题目范围 | 格子数 | 颜色 |
|------|---------|--------|------|
| 左宇翔 | 213-237 | 25个 | 红色 #FF6B6B |
| 周玉 | 238-262 | 25个 | 青色 #4ECDC4 |
| 彭逸 | 263-287 | 25个 | 蓝色 #45B7D1 |

## 安装步骤

1. **安装Node.js依赖**
```bash
npm install
```

2. **启动服务器**
```bash
npm start
```

3. **访问网站**
- 本机访问：http://localhost:5001
- 局域网访问：http://192.168.0.226:5001

## 使用说明

1. 打开网页后，先在下拉菜单中选择你的身份
2. 点击任何未完成的格子来标记完成（所有格子都可以点击）
3. 如果点错了，再次点击同一个格子可以取消
4. 别人完成的格子会显示完成者的颜色和名字，不能被其他人修改
5. 所有操作会实时同步到其他所有在线用户

## 文件结构

```
check_web/
├── server.js           # Node.js后端服务器
├── package.json        # 依赖配置文件
├── data.json          # 数据存储文件（自动生成）
├── README.md          # 本说明文件
└── public/            # 前端文件目录
    ├── index.html     # 主页面
    ├── style.css      # 样式文件
```

## 技术架构

- **后端**：Node.js + Express + Socket.io
- **前端**：原生 HTML + CSS + JavaScript
- **实时通信**：WebSocket (Socket.io)
- **数据存储**：JSON文件

## 注意事项

- 所有人都可以点击任何未完成的格子
- 只能取消自己完成的任务
- 不能修改别人完成的任务
- 格子的颜色区域仅用于视觉区分，不限制点击权限
- 数据自动保存在data.json文件中
- 服务器重启后数据不会丢失

## 端口说明

本服务使用 **5001** 端口，避免与现有的5000端口冲突。
